WEB-BASED SYSTEM for Pile Load Tests

The system contains a large number of case histories of axial drilled shast load tests from around the world. Information includes construction methed, pile geometry, soil investigation data, static load test results, and case-based drilled shast data. The system enables browsing, querying and collecting data.

Database Introduction

Information on drilled shafts was collected from different journals, literature, and load tests reports at various sites worldwide.Data is divided into seven categories:

  1. Site Data
  2. Soil Characteristics
  3. Shaft Information
  4. Design parameters
  5. Interpreted Capacity
  6. Shaft displacement
  7. Reference
